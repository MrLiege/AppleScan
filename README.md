# AppleScan

AppleScan — минималистичное и архитектурно‑чистое приложение для сканирования локальной сети и Bluetooth‑устройств. Проект построен на SwiftUI, Combine и модульной архитектуре с использованием Flow Coordinator для навигации и явных enum‑состояний для управления UI.

---

## Возможности

- Сканирование **LAN‑устройств** через `LanScanService`
- Сканирование **Bluetooth‑устройств** через `BluetoothService`
- Полноэкранный **Lottie‑loading overlay** с прогресс‑баром
- Обработка ошибок через состояние `.error`
- История всех сканирований с сохранением в `DatabaseService`
- Детальные экраны LAN и Bluetooth устройств
- Фильтрация по типу устройств и по дате
- Поиск по устройствам в истории
- Чистая навигация через FlowCoordinator
- Структурированный UI, разбитый на под‑вью и extension‑секции

---

## Архитектура

### Основные слои

- **View**  
  Чистые SwiftUI‑экраны, разбитые на под‑компоненты. Не содержат бизнес‑логики.

- **ViewModel**  
  Управляет состоянием экрана, подписками, сервисами и навигацией. Использует `@Published` и Combine.

- **Services**  
  - `BluetoothService` — поиск BLE‑устройств  
  - `LanScanService` — сканирование локальной сети  
  - `DatabaseService` — сохранение и загрузка сессий  

- **Flow Coordinator**  
  Управляет навигацией между экранами.  
  В проекте используются:
  - `ScanFlowCoordinator`
  - `HistoryFlowCoordinator`
